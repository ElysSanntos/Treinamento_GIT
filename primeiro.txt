-git      ==> verifica se o git está instalado na máquina
-git clone + paste ==> clona o repositorio criado no gitHub
-code .   ==> abre o visual code
-git status  ==>
-git add <nome do aquivo>  ==> versionando o arquivo
-git add * ==> versionando de todos os  arquivo
-git add . ==> versionando de todos os  arquivo
-git add -A ==> versionando de todos os  arquivo
-git commit  -m "o que foi alterado? Por que foi alterado?" ==> joga as alterações do arquivo para o repositorio local

-git checkout ==> mudar de branch 
-git checkout -b <nome da branch> ==> muda de branch e cria uma nova branch
-git push --set-upstream origin cadastro-usuario ==>Cria a branch no repositorio remoto e sincroniza
